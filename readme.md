# AI-Powered Fraud Detection System

A Machine Learning-based web application that analyzes transaction details and identifies potentially fraudulent financial transactions in real time.

## Live Application

https://mlfrauddetect.streamlit.app/

## Key Features

* Real-time fraud analysis
* Interactive Streamlit interface
* Machine Learning classification model
* Fast and reliable transaction assessment

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Joblib
* Streamlit
